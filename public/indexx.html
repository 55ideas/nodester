Nodester :: Opensource Node.js Hosting Platform

Nodester is an opensource Node.JS hosting platform with a RESTful API running on Amazon EC2. We are currently running Node v.0.3.4 and updating all of the NPM modules on a weekly basis. Git is required to push your updates to Nodefu. We do not currently have a full featured website as we are focusing on the API and the service. All we ask is that you be a good steward of this service and community.

You can access our source code repository at http://github.com/chrismatthieu/nodester

BASE URL: http://www.nodester.com

COUPON

Request Coupon :: POST

/coupon - creates coupon request for early access (pass in email) - Note: This resource does not use base api url
curl -X POST -d "email=chris@nodester.com" http://www.nodester.com/coupon

STATUS

Get Status :: GET

/status - Returns platform status and number of apps running
curl http://www.nodester.com/status

USER

Register User :: POST (Coupon is required.)

/user - creates user account (pass in user and password and email and id_rsa.pub string) - Note: This resource does not use base api url
curl -X POST -d "user=testuser&password=123&email=chris@nodester.com&rsakey=ssh-rsa AAAAB3NzaC1yc..." http://www.nodester.com/user

BASE URL: http://api.nodester.com

Delete User :: DELETE

/user - delete user account (requires basic auth)
curl -X DELETE -u "testuser:123" http://api.nodester.com/user

APP

Create Application :: POST

/apps - create nodejs app for hosting (requires basic auth and returns the port address required for use along with a git repo to push to)
curl -X POST -u "testuser:123" -d "appname=a&start=hello.js" http://api.nodester.com/app

Change Application :: PUT

/apps - update nodejs app for hosting (requires basic auth, appname, and starting page and returns the port address required for use along with a git repo to push to)
curl -X PUT -u "testuser:123" -d "appname=a&start=hello1.js" http://api.localhost:8080/app

Delete Application :: DELETE

/apps - delete nodejs app (requires basic auth and appname)
curl -X DELETE -u "testuser:123" -d "appname=test" http://api.nodester.com/app

GIT

Upon creating or changing your application via our API, you will receive a Git URL in our API response. Add a Nodester remote to your project as follows:

git remote add nodester the_url_returned_by_our_api

Finally push your updates to your new Nodester environment as follows:

git push nodester master

Also make sure that you update your project's port address with the new one assigned by Nodester.
