Opensource Node.js Hosting Services!

NodeFu is offering free experimental Node.js hosting services via our RESTful API running on Amazon EC2. We are currently running Node v.0.3.3 and updating all of the NPM modules on a weekly basis. Git is also required to push your updates to Nodefu. We do not currently have a full featured website as we are focusing on the API and the service. All we ask is that you be a good steward of this service and community. In other words, please don't try to hack our service and please reuse your port assignments when possible.

BASE URL: http://api.nodefu.com

USERS

/register - creates user account (pass in user and password)
curl -X POST -d "user=testuser&password=123" http://nodefu.com/register

/destroy - delete user account (requires basic auth)
curl -X DELETE -u "testuser:123" http://api.nodefu.com/destroy

APPS

/apps - create nodejs app for hosting (requires basic auth and returns the port address required for use along with a git repo to push to)
curl -X POST -u "testuser:123" -d "appname=a&start=hello.js" http://api.nodefu.com/apps

/apps - update nodejs app for hosting (requires basic auth, appname, and starting page and returns the port address required for use along with a git repo to push to)
curl -X PUT -u "testuser:123" -d "appname=a&start=hello1.js" http://api.localhost:8080/apps

/apps - delete nodejs app (requires basic auth and appname)
curl -X DELETE -u "testuser:123" -d "appname=test" http://api.nodefu.com/apps
